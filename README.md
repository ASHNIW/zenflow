### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/zenflow.git](https://github.com/ASHNIW/zenflow.git)
    cd zenflow
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run Web Version**
    ```bash
    npm run dev
    ```

4.  **Run Android Version**
    ```bash
    npm run build
    npx cap sync
    npx cap open android
    ```
